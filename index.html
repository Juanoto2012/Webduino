Bienvenido a Webduino

El Entorno de Desarrollo Integrado (IDE) para Arduino completamente funcional desde tu navegador. Sin instalaciones, sin drivers complejos, listo para programar.

Webduino utiliza tecnologías web modernas como Web Serial API y compilación en la nube para permitirte escribir código, verificarlo y flashearlo directamente a tu placa Arduino Uno desde Google Chrome o Microsoft Edge.

Guarda y organiza tus proyectos en la nube de forma segura sin perder una línea de código.

Añade dependencias (como Servo, DHT, etc.) y Webduino las resolverá automáticamente al compilar.

Requisitos Previos

Para usar Webduino correctamente, asegúrate de cumplir con los siguientes requisitos:

  • Navegador Compatible: Google Chrome, Microsoft Edge, o navegadores basados en Chromium recientes. (Safari y Firefox no soportan Web Serial API por defecto).
  • Hardware: Una placa Arduino Uno (o clon compatible con ATmega328P).
  • Cable USB: Un cable de datos USB para conectar tu placa a la computadora.

Nota: Al conectar tu placa por primera vez, tu navegador te pedirá permiso para acceder al puerto Serie. Debes otorgar este permiso para poder subir el código.

1. Conectar una Placa

El primer paso para empezar a trabajar es establecer la comunicación entre tu navegador y tu placa Arduino.

  1. Conecta tu Arduino a un puerto USB de tu computadora.
  2. En Webduino, haz clic en el botón superior que dice "Seleccionar Placa (Desconectado)" junto al ícono de USB.
  3. Aparecerá una ventana emergente del navegador. Selecciona el puerto correspondiente a tu placa (usualmente dice algo como "USB Serial", "CH340" o "Arduino Uno").
  4. Haz clic en "Conectar".

Si la conexión es exitosa, el botón se pondrá azul y dirá "Arduino Conectado".

2. Compilar y Subir Código

Webduino incluye un potente editor basado en Ace Editor con autocompletado para C/C++.

Verificar (Compilar)

Haz clic en el botón check Verificar. Tu código será enviado a nuestro motor de compilación en la nube (basado en Wokwi). Si hay errores de sintaxis, se mostrarán en la pestaña "Salida de Compilación" y se resaltarán con una X roja directamente en el editor de código.

Subir a Placa

Haz clic en play_arrow Subir a Placa. Esto ejecutará el proceso de compilación y, si es exitoso, iniciará el motor avrdude interno de JavaScript. La placa se reiniciará automáticamente y el código se escribirá en su memoria flash.

3. Monitor Serie

Para depurar tu código y leer valores de sensores, Webduino incluye un Monitor Serie integrado en el panel inferior.

  • Asegúrate de que la velocidad de baudios en la esquina inferior derecha coincida con la declarada en tu código (ej. Serial.begin(115200);).
  • El monitor funciona en tiempo real sin bloquear la interfaz.
  • Puedes enviar comandos a la placa escribiendo en la caja de texto y presionando Enter o el botón "Enviar".

4. Gestor de Librerías

Si tu proyecto requiere librerías externas (como pantallas LCD, sensores complejos o servomotores), no necesitas instalarlas manualmente en tu computadora.

  1. Haz clic en el ícono de libros (library_books) en la barra del explorador izquierdo.
  2. Se abrirá el Gestor de Librerías.
  3. Escribe el nombre de las librerías que necesitas, una por línea.

Ejemplo válido:

LiquidCrystal
Servo
DHT sensor library
Adafruit NeoPixel

Al hacer clic en Guardar, Webduino creará un archivo especial llamado libraries.txt. Durante la compilación, el servidor leerá este archivo e instalará automáticamente las dependencias antes de construir tu firmware.

5. Trabajar con Múltiples Archivos

A medida que tus proyectos crecen, es buena idea dividir el código en múltiples archivos (.ino, .cpp, .h).

  • Crear archivos: Usa el ícono note_add para añadir nuevas pestañas.
  • Subir desde tu PC: Usa el ícono upload para cargar archivos existentes desde tu disco duro directamente al editor de Webduino.

Al hacer clic en "Verificar" o "Subir", Webduino empaqueta de forma inteligente todos los archivos visibles en el explorador y los envía al compilador, por lo que los #include "mi_archivo.h" funcionarán a la perfección.

6. Sincronización en la Nube (Puter)

Webduino se integra con Puter.js para ofrecer almacenamiento persistente en la nube totalmente gratuito.

  1. Haz clic en el ícono de la nube (cloud) en la barra de actividad izquierda.
  2. Si es tu primera vez, se te pedirá iniciar sesión o crear una cuenta rápida de Puter.
  3. Una vez dentro, podrás Guardar tu entorno de trabajo actual (con todos sus archivos y librerías) bajo un nombre de proyecto.
  4. También verás una lista de tus proyectos guardados, con la opción de Cargar o Borrar en cualquier momento y desde cualquier computadora.
